Publish and document the project on GitHub
This project focuses on building interactive dashboards for Credit Card Transaction and Credit Card Customer analysis using Excel, PostgreSQL, and Power BI. Data is collected and cleaned in Excel, stored and managed in PostgreSQL, and visualized in Power BI. The dashboards provide insights into customer behavior, spending patterns, transaction trends, and key performance metrics to support data-driven decision making.
